# Security Policy

## Our Promise

We don't send anything to outside servers we or anyone else owns beyond your own AI subscription. This service does not establish any new outside API calls unless you the user direct it to do so. It is completely contained within your computer after you install it.

## Supported Versions

The latest minor version receives security updates.

## Reporting Security Issues

Please open a [GitHub Issue](https://github.com/josephwander-arch/workflow/issues) or email protipsinc@gmail.com.
